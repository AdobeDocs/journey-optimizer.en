# contain {#contain}

Checks if the second argument string is contained in the first argument string.

## Category

String

## Function syntax

`contain(<parameters>)`

## Parameters

* string

## Signature and returned type

`contain(<string>,<string>)`

Returns a boolean.

## Example

`contain("rowing is great", "great")`

Returns true.
